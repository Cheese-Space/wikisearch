# wikisearch
a cli for searching wikipedia articles

# requirements
this program requires python version 3 or higher, which is typicaly already installed on linux.  
On macOS, you can download python3 [here](https://www.python.org/downloads/macos/).

# flags
there are currently 5 flags:
1. --help: shows help message and exits
2. --Dutch: sets language to dutch
3. --Spanish: sets language to spanish
4. --Italian: sets language to Italian
5. --article: specify the article
